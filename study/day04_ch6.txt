day 04

웹 문서에서 스타일이란?
- HTML 문서에서 자주 사용하는 글꼴이나 색상, 정렬, 각 요소의 배치 방법과 같이 문서의 겉모습을 결정짓는 것
- ex) 텍스트 단락에서 줄간격을 조절하고 표의 테두리를 점선으로 바꾸는 것
* CSS를 이용하면 HTML로 작성한 내용은 그대로 두고 대상 기기에 맞게 CSS만 바꿔주면
같은 내용을 여러 기기에서 볼 수 있음

CSS 스타일 형식
기본형 -> 선택자 { 속성1: 속성값1; 속성2: 속성값2 }
- 맨 앞의 선택자 : 스타일을 어느 태그에 적용할 것인지
- 중괄호 사이에는 스타일 정보를 넣음
- 스타일 규칙 : 속성과 같이 하나의 쌍으로 이루어진 것이며, 세미콜론으로 구분해 여러개 지정할 수 있음

인라인 스타일
- 간단한 스타일 정보라면, 스타일 시트를 사용하지 않고 스타일을 적용할 대상에 직접 표시함
- ex) <p style="color: blue;">

내부 스타일 시트
- 스타일을 여러 곳에 적용할 때 씀
- 웹 문서 안에서 사용할 스타일을 같은 문서 안에서 정리한 것
- 스타일 정보는 웹 문서를 브라우저 화면에 표히사기 전에 결정해야 하므로
모든 스타일 정보는 <head> 태그 안에서 정의하고 <style>과 <style> 태그 사이에 작성함

외부 스타일 시트
- 디자인 일관성이 있도록 같은 스타일을 여러 웹 문서에 사용
-> 그때마다 웹 문서를 똑같은 내붑 스타일 시트로 만든다면 서버 공간과 문서 내려받는 시간 낭비
- 사이트를 제작할 땐 여러 웹 문서에서 사용할 스타일을 별도 파일로 저장해놓고 필요할 때마다 가져와서 사용
- 이렇게 따로 저장해 놓은 스타일 정보. *.css 의 파일 확장자 사용
기본형 -> <link rel="stylesheet" href="외부 스타일 시트 파일 경로">

타입 선택자
- 특정 태그를 사용한 모든 요소에 스타일을 적용
기본형 -> 태그명 { 스타일 규칙 }

클래스 선택자
- 특정 부분만 선택해서 스타일을 적용
- 클래스 이름 앞에 마침표(.)를 반드시 붙여야 함
기본형 -> .클래스명 { 스타일 규칙 }

클래스 스타일
- 클래스 선택자를 사용해 만든 스타일
- 클래스 스타일을 적용할 때는 태그 안에 class="클래스명" 으로 사용

id 선택자
- 클랙스 선택자와 마찬가지로 웹 문서의 특정 부분을 선택해서 스타일을 지정
- 클래스 선택자와의 큰 차이는 '클래스 선택자가 문서에서 여러번 적용할 수 있는 반면,
id 선택자는 문서에서 한번만 적용할 수 있음'
- 마침표(.) 대신 # 기호를 사용한다는 점만 제외하면 스타일을 정의하는 방법은 클래스 선택자와 같음
기본형 -> #아이디명 { 스타일 규칙 }

그룹 선택자
- 같은 스타일 규칙을 사용하는 요소들을 묶음
기본형 -> 선택자1, 선택자2 { 스타일 규칙 }
